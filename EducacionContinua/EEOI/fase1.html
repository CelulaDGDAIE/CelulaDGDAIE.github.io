Generating ideas

We all know how to write documents in our native language and we also know what expressions or vocabulary to use when we write. We can write reference letters, curriculum vitae, request and complain letters, however, do we know how to write them in English? It is very important to know what expressions, vocabulary and what format can be used in each case. The reason? In English  these documents /papers have to be written in a different way.  Let's see an example: 

  1. What make a CV difficult to read?
  2. How long can a curriculum vitae be?
  3. What is the purpose of the CV profile?
  4. How can your work experience be ordered?
  5. What are the elements a CV must contain?

Project 1:

Request letter and a complaint letter.

Description: Course participants write a request and a complaint letter following this type of documentation conventions.

Evaluation criteria:

The document must:

  1. Have clarity of purpose and thought.
  2. Convey accurate, relevant and complete information.
  3. Have short, concise and clear sentences.
  4. Have correct grammar and sentence structure.

Guidelines:

  1. The document must be written correctly using samples of the vocabulary and structures studied in this module. It must be written in Microsoft Word using font Arial size 12  and must be sent in an attachment. 
  2. Name your file as follows: Actn_LastName and FirstName. For example: Act1_VillanuevaMariaTeresa.
  3. Send your file, in Word format, via Activities tool in Eminus platform, no later than the due date set in the Course calendar.
